# Product Highlights (Target)

These are the product expectations we’re building towards (mobile first, iOS/Android friendly).

- 77 screens (from your provided list)
- 100% editable and customizable UI (code-based design system with tokens)
- Organized layers/components (Layout, Pages, Components, Theme)
- Global style guide (colors, type, spacing, components)
- Pixel-consistent spacing grid (8px base)
- Easy to extend (screen list drives routing; add new screen = 1 entry)

## Zoho-Inspired Product Behaviors (For Professional Grade)
This is inspiration only (not copying).

- Real-time attendance capture via mobile/web with restrictions (geo + IP).  
- Kiosk mode on a shared device (tablet) with identity verification (photo/face match) and device-specific login.  
- Biometric device integration to prevent buddy punching and false entries.  
- Attendance regularization flow (missed IN/OUT, incorrect times) with manager approval.  
- Break tracking and shift rules (paid/unpaid breaks, late/early, overtime).  
- Reports: daily/weekly/monthly summaries, late/early, muster roll, payroll export.  

## Notes On “Same As ThemeForest”
We cannot copy a ThemeForest template “same to same” without buying the license and using their provided files.
What we *can* do:
- Build an original UI with the same feature coverage and a similar UX pattern.
- If you buy a ThemeForest template and share it with your developer, they can integrate it legally.
