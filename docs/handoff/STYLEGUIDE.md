# UI Style Guide (Blue + Maroon)

This style is applied in `frontend/src/styles/global.css`.

## Colors
- Primary (Blue): `#0B4DBB`
- Accent (Maroon): `#7A0C2E`
- Background: warm light with subtle texture
- Text: near-black for readability

## Typography
- Headings: `Poppins` (Google Font)
- Body: `Inter` (Google Font)

## Layout
- Mobile-first, responsive
- App shell:
  - Sidebar navigation (collapses on small screens)
  - Topbar with current screen and Logout button

## Motion
- Page fade/slide on navigation
- Soft hover transitions
