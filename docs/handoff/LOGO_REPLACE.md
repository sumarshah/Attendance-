# Replace Logo

Current logo is a **placeholder** SVG using your brand colors (blue + maroon).

To replace with your real company logo:
1. Put your file here:
   - `frontend/src/assets/biotime-mark.svg`
2. Keep it square (best: 128x128 viewBox).
3. Rebuild frontend:
   - Local dev: `npm run dev`
   - Docker: `docker compose up -d --build frontend`
